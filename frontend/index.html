Условия использования

1. Общие положения

Настоящие Условия использования регулируют отношения между вами и "Спокойный рассвет"...

2. Регистрация аккаунта

Для использования сервиса необходимо зарегистрировать аккаунт, предоставив имя пользователя и Telegram username...

3. Роли пользователей

Система предусматривает следующие роли пользователей:

  • Пользователь - базовая роль при регистрации
  • Слушатель - пользователь с расширенными правами
  • Совладелец - пользователь с правами управления
  • Администратор - полные права управления системой
  • Владелец - высший уровень доступа

4. Восстановление пароля

В случае утери пароля вы можете восстановить его через Telegram. Для этого укажите ваш Telegram username в форме восстановления.

Администратор свяжется с вами в течение 24 часов в личные сообщения Telegram для восстановления доступа.

5. Конфиденциальность

Мы обязуемся защищать ваши персональные данные...
