Условия использования

1. Общие положения

Сообщество "Спокойный Рассвет" создано для взаимопомощи и поддержки. Мы стремимся создать безопасное пространство для всех участников.

2. Правила поведения

  • Уважайте других участников
  • Не разглашайте личную информацию
  • Избегайте оскорбительных высказываний
  • Соблюдайте конфиденциальность

3. Конфиденциальность

Мы защищаем ваши персональные данные и не передаем их третьим лицам.

4. Ответственность

Администрация оставляет за собой право блокировать пользователей, нарушающих правила сообщества.
