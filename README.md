# Чат Система

Многоуровневая чат система с системой ролей и расширенными возможностями управления.

## Функциональность

### Уровни доступа:

1. **Пользователь** - общение со слушателями, оставление отзывов
2. **Слушатель** - общение с пользователями, просмотр своих отзывов
3. **Администратор** - управление пользователями, просмотр статистики и чатов
4. **Совладелец** - все возможности администратора + технические уведомления, назначение ролей
5. **Владелец** - полный контроль над системой, управление совладельцами

### Основные возможности:

- Регистрация и аутентификация
- WebSocket чат в реальном времени
- Загрузка медиа файлов (фото, видео, аудио)
- Система стикеров
- Темы оформления (светлые и темные)
- Система уведомлений
- Модерация пользователей
- Статистика системы
- Логирование действий
